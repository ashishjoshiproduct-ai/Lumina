# Lumina

A web-based reading tool that helps you read actively. Upload a PDF, read it section by section, and (coming soon) ask an AI questions scoped to what you've read so far — so nothing gets spoiled.

## Status

Early MVP. Currently supports:
- Uploading any text-based PDF
- Splitting it into readable sections
- Navigating with Previous / Next
- Remembering your place per document (on your device)

## How to run it

Open `index.html` in any web browser. That's it — no install needed.
